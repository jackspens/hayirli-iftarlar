# Hayırlı İftarlar

Ramazan ayı için geliştirilmiş, İstanbul iftar ve imsak vakitlerini gösteren mobil uyumlu React uygulaması.

## Kurulum ve Çalıştırma

1. Projeyi klonlayın
2. `npm install` ile bağımlılıkları yükleyin
3. `npm run dev` ile yerel sunucuyu başlatın

Site ana branch'e kod pushlandığında GitHub Actions üzerinden otomatik olarak GitHub Pages'e deploy olmaktadır.
